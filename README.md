# 프로젝트 소개
2020년에 등장한 COVID-19 바이러스 감염증에 예방하기 위한 

방역 용품을 판매하는 BYE-RUS 쇼핑몰 사이트입니다.

## 역할
+ 임주혜(GOD) - 조장, 관리자 기능
+ 박용수(DRAGON) - 검색, 메인화면, 찜, 통합
+ 김반야(HALF) - 상품 목록, 상세페이지, 레이아웃 설계
+ 윤홍주(RED) - 장바구니, 결제창
+ 이정윤(HEART) - 마이페이지
+ 김희수(WATER) - 로그인, 게시판
